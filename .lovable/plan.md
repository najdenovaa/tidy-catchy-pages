
## Блок 1: Torque & Drag (Расчёт спуска колонны)

### Расчётный движок (`src/lib/torque-drag-calculations.ts`)
- **Soft-string модель** (метод Johancsik) — посегментный расчёт по траектории
- Режимы: Спуск (Trip In), Подъём (Trip Out), Вращение (Rotate), Бурение ротором, Бурение ГЗД
- Рассчитываемые параметры на каждой глубине:
  - **Эффективное натяжение** (Effective Tension) с учётом плавучести
  - **Истинное трение** (Drag / Friction Force) — боковая + осевая 
  - **Крутящий момент** (Torque) при вращении
  - **Боковая сила** (Side Force / Contact Force) на стенку
  - **Вес на крюке** (Hook Load) для каждого режима
  - **Зазор колонны** (Clearance) — минимальное расстояние до стенки
- Входные данные: профиль скважины (trajectory), вес колонны (кг/м), OD/ID, коэфф. трения (сталь-сталь/сталь-порода), WOB, RPM
- Формулы на основе buoyancy factor, incremental drag, incremental torque

### UI: новая вкладка «T&D» (`src/components/TorqueDragSection.tsx`)
- **Ввод параметров**: коэфф. трения (раздельно для ОК и открытого ствола), вес трубы, WOB, RPM
- **Таблица сводки**: вес на крюке, макс. момент, макс. боковая сила, мин. вес на долоте
- **Графики (Recharts)**:
  1. Натяжение vs глубина (Trip In / Trip Out / Rotate)
  2. Крутящий момент vs глубина
  3. Боковая сила vs глубина
  4. Зазор колонны vs глубина

## Блок 2: Пеноцементирование (`src/lib/foam-cement-calculations.ts`)

### Расчётный движок
- Расчёт качества пены (Foam Quality, 20–80%) по глубине
- Начальный и конечный объём пеноцемента
- Расход подачи азота (N₂ rate)
- Обратное давление (Back Pressure)
- Плотность пеноцемента по глубине (с учётом сжатия газа — закон Бойля)
- ECD пеноцемента в затрубье

### UI: секция в существующей вкладке или новая вкладка «Пена»
- Ввод: базовая плотность цемента, целевое качество пены (%), обратное давление, температура
- Таблица: глубина → плотность → качество пены → объём N₂
- Графики: плотность по глубине, качество пены по глубине, расход N₂
